# Lånekalkulator

Enkel boliglån-kalkulator for 2 personer — beregner eierandel, månedlig kostnad, rentekostnader og skattefradrag.

## Oppsett

Ingen. Åpne `index.html` i nettleseren. Ferdig.

Ingen npm, ingen build-steg, ingen avhengigheter å installere. Alt er i én HTML-fil med innebygd CSS og JavaScript. Eneste eksterne ressurs er [Chart.js](https://www.chartjs.org/) som lastes fra CDN.

## Funksjoner

- Effektiv rente med korrekt månedlig renteutledning
- Annuitetslån og serielån
- To personer med individuell egenkapital og lånebeløp
- Automatisk beregning av eierandel
- Skattefradrag på rentekostnader (22 %)
- Visuell eierfordelingsbar med mangler/over-advarsel
- Grafer: årlig oversikt, restgjeld og per-person sammenligning
- Årlig nedbetalingsplan som tabell
